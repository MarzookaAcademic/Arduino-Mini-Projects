# LED Blink

A simple Arduino project that blinks an LED on and off every second.

## Components
- Arduino UNO
- LED
- 220Ω resistor
- Breadboard, jumper wires

## Wiring
- LED anode → digital pin 13  
- LED cathode → resistor → GND

## How to Run
1. Open LED_Blink.ino in Arduino IDE  
2. Select Board: Arduino UNO  
3. Select correct Port  
4. Click Upload  
5. Watch LED blink every second!
